# 2Ejercicio  
Titulo del proyecto: Ejercicio 2- Suma de digitos hasta que el resultado sea de un solo digito
Proposito del proyecto: Returnar el resultado de la suma de los digitos de un numero hasta que sea de un solo digito
Version o fecha: 28/05/2020
Como iniciar este proyecto: Ejecutar el main de la clase
Autores: Emilio Abraham Arreola Cruz 19161240
Instrucciones para el usuario:
Para prueba del programa se debe de ejecutar el main de la clase y ingresar un numero.
Solucion: 
Primero se introduce el numero y por medio de operaciones aritmeticas se van separando los digitos, para despues
 irlos sumando, despues se hizo otro ciclo para el caso en el que la suma sea de dos digitos
